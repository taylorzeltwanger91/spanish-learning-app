Connect Your Homework Folder

Lengua can automatically import vocabulary from Google Docs in your Drive folder. Follow these steps to set up the connection. The whole process takes about 5 minutes and uses Google's free tier.

1

Create a Google Cloud Project

Go to the Google Cloud Console and create a new project. Give it any name (e.g. "Lengua"). The free tier is all you need.

2

Enable the Google Drive API

  1. In your project, go to APIs & Services → Library
  2. Search for "Google Drive API"
  3. Click it, then click Enable
Make sure you're in the correct project when enabling the API.
3

Create an API Key

  1. Go to APIs & Services → Credentials
  2. Click + Create Credentials → API Key
  3. Copy the key (starts with AIza...)
Optional: Under "Restrict key", you can limit it to only the Google Drive API for extra security. If you restrict it, make sure Google Drive API is in the allowed list.
4

Create a Homework Folder in Google Drive

In Google Drive, create a new folder for your Spanish homework documents. Put your vocabulary docs inside this folder.

5

Share the Folder Publicly

  1. Right-click the folder → Share
  2. Under "General access", change to Anyone with the link
  3. Set the role to Viewer
  4. Click Done
This is required because Lengua uses an API key (not OAuth sign-in) to read your files. Only "Viewer" access is needed.
6

Copy the Folder ID

Open the folder in Drive. The URL will look like:

https://drive.google.com/drive/folders/1aBcDeFgHiJkLmNoPqRsTuVwXyZ

The Folder ID is the long string after /folders/. Copy it.

7

Connect in Lengua

  1. Open Lengua and go to the Drive page
  2. Paste your Folder ID and API Key
  3. Click Test Connection to verify everything works
  4. Click Connect & Sync to import your vocabulary

Troubleshooting

"API key not valid" / 400 Error

Double-check that you copied the full API key. It should start with AIza and be about 39 characters long. Make sure there are no extra spaces.

"Folder not found" / 404 Error

Verify the Folder ID is correct (the string after /folders/ in the Drive URL). Also make sure the folder sharing is set to "Anyone with the link".

"Forbidden" / 403 Error

This usually means one of:

  • The Google Drive API is not enabled in your Cloud project
  • Your API key is restricted and doesn't include the Drive API
  • The folder is not shared publicly (needs "Anyone with the link")

"Rate limit exceeded" / 429 Error

You've hit Google's API rate limit. Wait a minute or two, then try again. This is uncommon with normal usage.

Files aren't being imported

Make sure your documents use a supported format:

hola - hello
buenos días - good morning
hablar - to speak

Supported separators: -, =, : between Spanish and English words.
